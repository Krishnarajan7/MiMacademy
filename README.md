# 🎓 MiM Academy

[![Website](https://img.shields.io/website?url=https%3A%2F%2Fmimacademy.org)](https://mimacademy.org) [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE) [![React](https://img.shields.io/badge/React-18.2.0-61DAFB?logo=react)](https://reactjs.org/) [![Vite](https://img.shields.io/badge/Vite-5.0-646CFF?logo=vite)](https://vitejs.dev/)

**MiM Academy** is a modern, responsive, React-powered platform delivering **personalized tutoring and coaching** through online and in-person sessions. Empowering students and professionals with top-tier education and guidance! 🚀

---

## 🌐 Live Site

👉 **[Visit MiM Academy](https://mimacademy.org)**

---

## 🧩 What It Does

- 📘 **Academic Tutoring**: Expert-led sessions in **Physics**, **Chemistry**, **Biology**, **Math**, and **Computer Science**.
- 👨‍💼 **Professional Coaching**: Personalized 1-on-1 and group-based coaching for career growth.
- 🎥 **Flexible Learning**: Choose between **video-based** or **in-person** sessions.
- 📅 **Easy Scheduling**: Seamless process to reach out and schedule sessions.

---

## ⚙️ Tech Stack

- **Frontend**: React + Vite for a fast, modern UI.
- **Styling**: Custom CSS3 (with optional frameworks).
- **Deployment**: Static hosting via Netlify, Vercel, or GitHub Pages.
- **No Backend**: Purely frontend, lightweight, and efficient.

## 📁 Project Structure

```bash
MiMacademy/
├── public/            # Static assets (images, fonts, etc.)
├── src/
│   ├── assets/        # Images, icons, and other resources
│   ├── components/    # Reusable React components
│   ├── pages/         # Page-specific components
│   ├── App.jsx        # Main app component
│   └── main.jsx       # Entry point
├── index.html         # HTML template
├── package.json       # Project config and dependencies
└── vite.config.js     # Vite configuration
```
 ## 🚀 Getting Started
Clone the Repository
```
git clone https://github.com/Krishnarajan7/MiMacademy.git
```
```
cd MiMacademy
```
## Install Dependencies
```
npm install
```
## Run the Development Server
```
npm run dev
```
Your app will be live at http://localhost:5173.

## Build for Production
```
npm run build
```
## Preview the build locally:
```
npm run preview
```
